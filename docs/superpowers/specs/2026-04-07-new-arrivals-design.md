## Overview

New Arrivals is a daily puzzle game where you play as a video store clerk sorting 16 VHS tapes into 4 hidden genre categories on a 3D shelf. Think NYT Connections meets an 80s Blockbuster. Built with Three.js, deployed on Netlify, mobile-first.

**Share URL:** game.vhsgarage.com

---

## Architecture

### Tech Stack

| Layer | Choice |
|-------|--------|

## Game Rules & Economy

### Starting State
- 16 VHS tapes in a 4x4 grid (animated entrance — tapes spin in from off-screen)
- Player starts with **$10.00 daily wage**
- Timer starts on first interaction

### Core Loop
1. Examine 16 pixelated VHS covers
2. Drag 4 tapes into a row (staging a guess)
3. Tap **"Shelve It"** to submit
4. **Correct:** Tapes lock into their row, category name + color revealed. Rows fill top (easiest) to bottom (hardest).
5. **Wrong:** Tapes bounce back. **-$1 penalty.** VHS tracking distortion flash.
6. Repeat until all 4 categories solved or player hits $0.

### Hint System — "Uncover"
- All 16 covers start pixelated (~32px mosaic)
- **Tap** any tape: spins into full-screen lightbox showing pixelated cover + movie title
- **Uncover ($1):** Clears pixelation to full-res TMDB poster. Film-projector animation.
- **Max 5 uncovers per game.** After 5, button shows "Out of Focus."
- Uncovered tapes show subtle glow on shelf.

### Scoring
```
Final Wage = $10 - (wrong guesses x $1) - (uncovers x $1) - time penalty
Time penalties: <2min = $0, 2-5min = -$1, >5min = -$2
Minimum = $0
```

### End States
- **All 4 rows solved:** Victory screen with final wage
- **$0 reached:** "The Store Closed Early" — show remaining answers

## Visual Design

**Aesthetic:** "Fluorescent-lit VHS paradise" — warm CRT glow, scan lines, wood-panel shelving, chunky geometry. Friday night 1987 video store vibes.

### Scene
- **Camera:** Fixed orthographic or tight perspective. No orbit/zoom/scroll.
- **Shelf:** 4 wooden planks with side panels. Warm brown wood.
- **VHS Boxes:** Chunky rectangles (~1:1.6 aspect), poster texture on front, slight bevel, idle wobble.
- **Lighting:** Warm overhead fluorescent (slight flicker), soft ambient, slight bloom.
- **Background:** Dark store interior, neon "OPEN" sign glow.

### Color Palette
```
--shelf-wood: #8B6914       --neon-pink: #FF6B9D
--shelf-dark: #5C4411       --neon-blue: #00D4FF
--bg-store: #1A1A2E         --crt-green: #39FF14
--category-easy: #4CAF50    --category-hard: #2196F3
--category-medium: #FFC107  --category-devious: #9C27B0
```

### Typography
- **Headers:** "Press Start 2P" — chunky VHS-label style
- **Body/UI:** "Space Mono" — monospaced, period-appropriate
- **Category reveals:** All-caps with glitch/flicker animation

## Audio Design (Tone.js)

All synthesized at runtime. Key events:

| Event | Sound |
|-------|-------|
| Tape land (start) | Low-mid thud, staggered, slight pitch variation |
| Tap to inspect | Upward "zip" — saw wave pitch bend, 100ms |
| Return to shelf | Soft "plop" — sine wave pitch drop |
| Drag start | Click — noise burst, 20ms |
| Drop in place | "Clack" — noise + square wave, 50ms, reverb |
| Correct guess | Ascending 4-note major arpeggio, warm sine/triangle |
| Wrong guess | Dissonant buzz — detuned saws + VHS tracking crackle |
| Uncover hint | Projector whir → clean "ding" |
| Penalty | Reverse cash register — descending pitch |
| Game win | Synth fanfare — 2s major chord swell |
| Game loss | CRT power-down — pitch drops to silence, 1.5s |
| Share generated | Polaroid "click-whir" |

## Interaction Model (Mobile-First)

- **Tap:** Inspect mode (lightbox)
- **Tap+hold+drag:** Lift and follow thumb, snap to drop zones
- **Tap vs drag threshold:** >10px movement = drag, otherwise tap
- **Raycasting:** Three.js `Raycaster` on touch/mouse events
- **Drop zones:** Invisible planes aligned with each shelf row, snap to nearest valid slot
- **Staging:** Player drags 4 tapes into any row, "Shelve It" checks that group

## Poster Pipeline

### Build-Time Script (`scripts/fetch-posters.js`)
- Reads `public/puzzles.json`, collects all unique `tmdb_id` values
- Fetches poster URL from TMDB API
- Downloads full-res poster to `public/posters/{tmdb_id}.jpg`
- Generates pixelated version (32x48 downscale → nearest-neighbor upscale) to `public/posters/{tmdb_id}_pixel.jpg`
- Uses `sharp` for image processing

### Runtime Texture Loading
- Three.js `TextureLoader` loads pixelated version on game start
- On "Uncover," swaps to full-res with shader transition
- All 16 full-res posters preloaded in background for instant uncover

## Puzzle Content

5 starter puzzles with themes: "Tuesday Restock," "Friday Night Rush," "Slow Wednesday," "New Release Wall," "Saturday Morning Shift." Each has 4 categories at difficulty 1-4 with overlap traps. Era range primarily 1975-1995.

Daily rotation: puzzle ID = today's date. If no puzzle for today, cycle via `daysSinceEpoch % puzzleCount`.

## State Management

### localStorage Keys
```
newArrivals_onboarded     — boolean
newArrivals_today         — ISO date string
newArrivals_state         — JSON: { solvedCategories, wrongGuesses, hintsUsed, uncoveredIds, startTime, wage, completed }
newArrivals_stats         — JSON: { totalGames, totalWages, bestWage, currentStreak, maxStreak, history[] }
```

### Daily Reset
- Compare `newArrivals_today` to current date
- If different: clear state, set new date, load new puzzle
